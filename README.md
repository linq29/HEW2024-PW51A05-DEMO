本リポジトリは、  
個人制作作品 **「十二支にまつわる神社探し」** の DEMO バージョンです。

ユーザーの生年から干支を計算し、  
その干支にゆかりのある神社や、  
気になるご利益に応じて縁のある神社をおすすめするWebサイトです。


## 概要

- テーマ：十二支をテーマとした神社探しサイト  
- 制作形式：自主制作
- 展示時期：2025年4月 ／ 2026年2月（リニューアル）


## 使用技術

- HTML / CSS  
- JavaScript（データ処理・画面制御）  
- Photoshop / Illustrator（素材制作）


## 画像・資料について

本サイト内で使用している画像・イラスト・資料は、  
**すべて授業課題としての学習目的の範囲内で引用**しています。
