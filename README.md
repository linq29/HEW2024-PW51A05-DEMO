# HEW2024-PW51A01-DEMO
**GitHubPages ▶︎ https://linq29.github.io/HEW2024-PW51A05-DEMO/**

本リポジトリは、進級制作展（HAL Event Week）にて展示した  
個人制作作品 **「十二支にまつわる神社探し」** の DEMO バージョンです。

ユーザーの生年から干支を計算し、  
その干支にゆかりのある神社や、  
気になるご利益に応じて縁のある神社をおすすめするWebサイトです。

※ 検索避けのため、リポジトリ名には課題番号を使用しています。


## 概要

- 制作内容：十二支をテーマとした神社探しサイト  
- 制作目的：Web 構造設計および2024年10月〜2025年3月に習得した技術要素の統合  
- 制作形態：進級制作（個人制作）


## 使用技術

- HTML / CSS  
- JavaScript（データ処理・画面制御）  
- Photoshop / Illustrator（素材制作）


## 制作時期

- 提出・展示時期：2025年4月

## 備考

- AI補助あり（chatGPT / Gemini / Grok）
- 本リポジトリは GitHub Pages 上での公開を前提とした  
**静的 Web サイト**として構築しています。  
展示時はサーバーサイドのデータベースや SQL は使用せず、  
データベースを想定した構造で Excel（.xlsx）にてデータ設計を行い、  
最終的に JSON 形式へ変換したうえで、  
クライアントサイドの JavaScript により  
データの管理・処理を行っています。

- 本リポジトリは DEMO バージョンとして公開しており、  
本来はデータベースと連動することを想定した設計部分については、  
静的ファイルによる遷移およびデータ管理で代替しています。
